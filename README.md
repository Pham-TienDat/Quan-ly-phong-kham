# Quản lý phòng khám

## Nghiệp vụ: Quản lý hồ sơ bệnh nhân
- Thêm / sửa / xóa bệnh nhân
- Tìm kiếm theo tên, SĐT, BHYT
- Xem lịch sử khám

## Nghiệp vụ: Thu ngân và thanh toán
- Xem danh sách các lượt khám chưa được thanh toán
- Tự động tính tiền dựa trên đơn thuốc
- Tạo hoá đơn
- Thanh toán
- Cập nhật trạng thái hoá đơn

## Công nghệ
- Frontend: HTML, CSS, JS
- Backend: PHP (PDO)
- Database: MySQL / MariaDB

## Cách chạy
1. Import database.sql
2. Chạy XAMPP (Apache + MySQL)
3. Truy cập: http://localhost/clinic/
